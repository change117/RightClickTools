# Quick Start - Build & Desktop Instructions

## What You Need to Do

You already have the RightClickTools folder on your Windows computer.

### Step 1: Open RightClickTools Folder
Find your RightClickTools folder and open it.

### Step 2: Run the Build Script
**Double-click: `INSTALL-COMPLETE.bat`**

That's it! The script will:
- ✓ Check for dependencies
- ✓ Install anything missing
- ✓ **BUILD the project**
- ✓ Copy the EXE to your Desktop
- ✓ Start the installer

### Step 3: Watch For Desktop File
When the script finishes, look on your Desktop for:
```
C:\Users\YourUsername\Desktop\RightClickTools.exe
```

You'll be able to see it with Everything search tool immediately.

### Step 4: Install Features (Automatic)
The installer dialog will pop up automatically. Just click **Install** to add all the right-click features.

---

## Desktop Location

The finished EXE will be:
```
C:\Users\YourUsername\Desktop\RightClickTools.exe
```

Replace `YourUsername` with your actual Windows username.

For example, if your username is "John", it will be:
```
C:\Users\John\Desktop\RightClickTools.exe
```

---

## That's All!

Double-click `INSTALL-COMPLETE.bat` in your RightClickTools folder and wait. Everything else happens automatically. The finished product will be on your Desktop.
